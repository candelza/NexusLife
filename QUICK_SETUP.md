## 🎯 วิธีใช้งานด่วน

### 1. รันสคริปต์อัตโนมัติ (แนะนำ)

```bash
# ติดตั้ง dependencies
npm install @supabase/supabase-js

# รันสคริปต์
node run_mockup_setup.js
```

### 2. ใช้ SQL Script

1. เปิด Supabase Dashboard
2. ไปที่ SQL Editor
3. คัดลอกเนื้อหาจาก `insert_mockup_prayers.sql`
4. รันสคริปต์
